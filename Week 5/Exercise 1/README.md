# Simple Potentiometer Reading with arduino Nano

Hi, this example shows how to read potentiometer reading on Analog Pin

Components needed: </br>
Arduino Nano/Uno </br>
1 x 10k ohm Potentiometer </br>
1 x breadboard </br>
3 x jumper male to male wire </br>
